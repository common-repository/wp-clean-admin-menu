WP Clean Admin Menu optimize dashboard experience, by removing unnecessary menu items, and managing menu visibility based on user roles.

== Description ==

WP Clean Admin Menu is a lightweight and user-friendly plugin that allows you to easily reorganize and customize your WordPress dashboard menu. By removing unnecessary or unused menu items, you can create a clean and intuitive dashboard that aligns with your specific needs and preferences.

If you need to see those menu items, just toggle those menu items with 'Toggle Menu Items' button from the left menu.

== WP Clean Admin Menu Pro ==
With the new [WP Clean Admin Menu Pro](https://demotolive.com/wordpress-plugins/wp-clean-admin-menu-pro/), you can control menus based on user role.

= Features: =

* Easy to Use
* Simplify the User Interface
* Hide Unused Menu Items
* Role-based Menu Management
* Hide Admin Menus & Sub-Menus

== Installation ==

1. Download and activate the plugin.
2. Browse to `Settings > WP Clean Admin Menu` to configure.
